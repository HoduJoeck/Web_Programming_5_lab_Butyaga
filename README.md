Я использовал localhost Apache2.4 httpd и php-8.5.2 на момент 19.01.26.
Файлы из каждой папки: index.html(.php), source.php, ... - находились изначально в папке htdocs. 
Каждая папка - своя отдельная версия сайта.

Чтобы включить/перезапустить/выключить сервер Apache я использовал батник, где включалась и выключалась служба "Apache2.4" с админскими правами.
Перезапуск требовался часто из-за изменения настроек php.ini и httpd.conf.

Из интересного, если зайти в код bat-файла можно найти еще одну службу, которая относится к локальному серверу postgreeSQL.
Это мне нужно было при попытке использовать Open Server с postgreSQL, установленной отдельно и гораздо новее, чем те, которые предлагались в Open Server.
К сожалению, я так и не разобрался как заставить работать php с pgsql / pdo_pgsql, и через Apache http, и через Open Server.
